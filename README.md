# Arrhythmia Early Prediction

A deep learning project for detecting and predicting cardiac arrhythmias using ECG signals.

## Features

- ECG preprocessing
- Beat segmentation
- CNN baseline
- CNN + BiLSTM + Attention
- Transformer-based ECG classifier
- Early arrhythmia prediction
- Performance comparison
- Streamlit dashboard

## Dataset

- MIT-BIH Arrhythmia Database

## Tech Stack

- Python
- TensorFlow
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
